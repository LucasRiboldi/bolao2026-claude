4. Acessibilidade

Quick reference para verificações manuais e WCAG 2.1 AA.

4.1 — Focus visible

Pressione Tab para ver o outline dourado em torno de elementos focáveis.

Link

4.2 — Contraste (texto vs fundo)

WCAG 2.1 AA exige ≥ 4.5:1 para texto normal e ≥ 3:1 para texto grande (18pt+).

text vs bg: 14.5:1 — AAA ✓
text vs surface: 13.1:1 — AAA ✓
text-muted vs surface: 6.4:1 — AAA ✓
gold vs surface: 6.9:1 — AAA ✓
success-text vs success-soft: 4.6:1 — AA ✓
danger-text vs danger-soft: 4.7:1 — AA ✓

4.3 — Reduced motion

O app respeita prefers-reduced-motion: reduce globalmente (em src/index.css): todas as animações caem para 0.01ms. Teste no SO: Settings → Accessibility → Display → Reduce motion.

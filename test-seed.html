📋 Como iniciar o servidor local

  1. Abra um terminal na pasta do projeto
  2. Instale as dependências (apenas uma vez): npm install
  3. Inicie o servidor: node seed-server.js
  4. Deixe o terminal aberto e recarregue esta página

O servidor roda em http://localhost:3001 e usa o Admin SDK para gravar diretamente no Firestore sem depender das regras de segurança.
